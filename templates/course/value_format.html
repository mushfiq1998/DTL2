Note: floatformat when used without an argument,
rounds a floating point number to one decimal place.

n1|floatformat: {{n1|floatformat}}

n2|floatformat: {{n2|floatformat}}

n3|floatformat: {{n3|floatformat}}

Note: If used with a numeric integer argument,
floatformat rounds a number to that many decimal places.

n1|floatformat:3: {{n1|floatformat:3}}

n2|floatformat:3: {{n2|floatformat:3}}

n3|floatformat:3: {{n3|floatformat:3}}

Note: floatformat is particularly useful is passing 0 as the argumment,
which will round the float to the nearest integer.

n1|floatformat:'0': {{n1|floatformat:'0'}}

n2|floatformat:'0': {{n2|floatformat:'0'}}

n3|floatformat:'0': {{n3|floatformat:'0'}}

Note:If the the argumment passed to floatformat is negative, it will round a number
to that many decimal places. but only if there is a decimal part to be displayed

n1|floatformat:'-3': {{n1|floatformat:'-3'}}

n2|floatformat:'-3': {{n2|floatformat:'-3'}}

n3|floatformat:'-3': {{n3|floatformat:'-3'}}
